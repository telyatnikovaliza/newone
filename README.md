# Crystal Quest - Platformer Game

Современная игра-платформер, созданная на чистом JavaScript с использованием Canvas API.

## Описание

Crystal Quest - это увлекательная игра в жанре платформер, где игрок должен собирать кристаллы, избегать врагов и проходить три уровня возрастающей сложности.

## Особенности

- **3 уровня** с возрастающей сложностью
- **Система жизней** - 3 сердца
- **Сбор кристаллов** с подсчетом очков
- **Умные враги** - патрулируют платформы
- **Физика движения** - гравитация и плавное управление
- **Современный UI** - красивое меню и HUD
- **Адаптивный дизайн** - работает на разных разрешениях

## Управление

- **A** или **←** - Движение влево
- **D** или **→** - Движение вправо
- **W** или **Space** - Прыжок

## Технологии

- HTML5 Canvas API
- Vanilla JavaScript (ES6+)
- CSS3 с градиентами и анимациями

## Структура проекта

```
├── index.html          # Главный HTML файл
├── css/
│   └── style.css       # Стили игры
├── js/
│   ├── main.js         # Точка входа
│   ├── Game.js         # Основной класс игры

│   ├── Platform.js     # Класс платформы
│   ├── Enemy.js        # Класс врага

│   ├── InputHandler.js # Обработка ввода
│   └── Collision.js    # Система коллизий
└── README.md           # Документация
```

## Запуск

1. Склонируйте репозиторий
2. Откройте `index.html` в браузере
3. Нажмите "Начать игру"

Для GitHub Pages: просто загрузите файлы в репозиторий и активируйте GitHub Pages в настройках.
